```
- your-theme
  - [dir] dot-config
  - [dir] wallpaper
    - static.png
    - live.mp4
  - apply.sh
  - desc.txt
  - icon.txt
  - pkiller.sh
  - theme.txt
```

### `dot-config`
Contain files that will be copied to `~/.config` (backups will be made before overwriting)

### `wallpaper`
Theme-specific wallpapers

### `apply.sh`
Theme-specific apply script (refer to the [explanation](https://github.com/WaylakeAnimations/lakeside-theme/blob/main/explanation.md))

### `desc.txt`
Theme description, can be read if the selector you use is capable enough 

### `icon.txt`
Icon theme specified by the theme

### `pkiller.sh`
Theme-specific pkill script (refer the [explanation](https://github.com/WaylakeAnimations/lakeside-theme/blob/main/explanation.md))

### `theme.txt`
GTK theme specified by the theme
